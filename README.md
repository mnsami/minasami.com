# My Personal blog

My personal blog is built using:

- Jekyll
- Twitter bootstrap.

## Requirements

- Ruby
- RubyGems

## Usage

- To install, run `make install`
- To build, run `make build`
- To test, run `make test`
- To run locally, run `make run`
